# CODSOFT Data Science Internship

## Overview

This repository contains all projects completed during the CodSoft Data Science Internship Program. These projects demonstrate skills in data preprocessing, exploratory data analysis (EDA), machine learning model development, evaluation, visualization, and deployment.

---

## Projects Included

### 1. Titanic Survival Prediction

Predict passenger survival using historical Titanic passenger data.

#### Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Classification
- Model Evaluation
- Flask Web Application

#### Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Flask
- Matplotlib
- Seaborn

---

### 2. Movie Rating Prediction

Predict movie ratings using machine learning based on movie attributes and historical rating data.

#### Features
- Data Analysis and Visualization
- Data Preprocessing
- Feature Engineering
- Regression Modeling
- Performance Evaluation
- Interactive Prediction System

#### Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

---

### 3. Iris Flower Classification

Classify Iris flower species using sepal and petal measurements.

#### Features
- Data Exploration
- Data Visualization
- Classification Algorithms
- Model Training and Testing
- Performance Analysis
- Real-Time Prediction

#### Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
